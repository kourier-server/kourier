# Setting Up VSCode for C++/Qt Development
I use VSCode for C++/Qt development. To streamline the setup of new machines, I wrote this guide where I set up VSCode on a Debian-based machine with debugging support for many Qt classes both locally and on dev containers. The created dev containers contain unsanitized and sanitized versions of Qt and its dependencies.

You may find this guide useful to set up VSCode on your Linux distribution of choice or even on another OS.

Before [creating](DevContainers/README.md) the VSCode dev containers, I [configure](DevEnvSetup/README.md) my local environment for C++/Qt development.
